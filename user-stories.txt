MVP USER STORIES

-- Login/Registration --

As a user I want to be able to insert a username and password to log in to the application

As a user I want to be able to create a username and password to log in to the application

As a user I want to be able to insert my email in the Registration process so if i forget my password i have a place for them to be sent

As a user I want to be able to edit my password or have a forget my password if I need a new password

-- Channels --

As a user I would like to be able to view my channels so I may select which channel I want to be in.

As a user I would like to be able to create or remove channels so I have only the channels I want.

As a user I would like to be able to rename the channel incase I mispell or create a wrong title.

As a user I would like to be able to reorder the channels to specify the most important.

As a user I want to be able to view who is in the channel.

-- Profiles --

As a user I want to be able to edit my info, if anything changes

As a use I would like to be able to upload a profile picture

-- Group Chat/ Direct Chat --

As a User I want to be able to post a message to a chat window, so it can be displayed on the chat window.

As a user I want to be able to specify where to post my messages, depending on where I want my message to go

-- Chat Window --

As a user I want to be able to display my posted messages, so others can see my post.

# User Statements For Future Features

## Searching

- as a user i want to be able to search for different channels that exist
- as a user i want to be able to search for different users that are on the app
- as a user I want to be able to search topics

## Notifications

- As a user, I want to be informed when someone sends me a message.
- As a user, I want to be able to know when someone invites me to a channel
- As a user, I want to know when someone has left the channel(within the channel)

## Invite

- As a user, I want to be able to invite someone into a channel

## Responsive

- As a user, I want to be able to resize the page.
- As a user, I want to be able to see a mobile view of the app

## Time Stamps

- As a user, I want to be able to see actual time in which a message was sent
- As a user, I want to see how long ago a message was sent

## Tagging

- As a user, I want to be able to tag someone so that the person I want to reach is notified immediately that I specifically mentioned them in a message.

## Pinning

- As a user I want to be able to pin an important message.(personal)
- As a user, I want to be able to pin a specific message so that everyone within the channel is able to access it easily without scrolling through endless messages.

## Dark Mode/ Themes

- As a user, I want to customize the appearance of the app so I do not strain my eyes.

- As a user I want to change the colors of the text, background, sidebar and other things.

## Status

- As a user, I want to be able to see who is online so that I know they are available, away or DND on the app.

## Message Count

- As a user, I want to be able to see how many messages I have to stay organized

## Emojis

- As a user, I want to be able to add silly faces, gifs, and other animations so that it is more personable and engaging.
- As a user, I want to see a list of all the options of the emojis that I am able to use in my messages.
